## ğŸ“ Quick Reference

### Basic Syntax

```zexus
# Variables
let name = "Zexus"
const VERSION = "0.1.0"

# Functions
action greet(name: string) -> string {
    return "Hello, " + name
}

# Entities (structs)
entity User {
    name: string,
    age: integer,
    active: boolean = true
}

# Contracts (smart contracts)
contract Token {
    persistent storage balances: Map<Address, integer>
    
    action transfer(to: Address, amount: integer) {
        # Implementation
    }
}

# Security
protect(sensitive_function, {
    rate_limit: 100,
    auth_required: true
}, "strict")

verify(user_authenticated) {
    log_error("Unauthorized access")
    block_request()
}

# Reactive State
watch variable {
    print("Variable changed!")
}

# Async/Await
async action fetch_data() {
    let result = await api_call()
    return result
}
```

### Key Features at a Glance

- **Types**: integer, float, string, boolean, list, map, set, entity, contract
- **Control Flow**: if/elif/else, while, for, match, try/catch
- **Functions**: action (named), lambda (anonymous), async
- **Security**: protect, verify, restrict, audit, sandbox
- **Blockchain**: contract, persistent storage, transactions, events
- **Performance**: native, inline, simd, gc, buffer
- **Reactive**: watch, events, streams
- **Modules**: import, export with access control
- **Testing**: test_mode, mock_dependency, inject

## ğŸ¯ Key Concepts

### 1. Policy-as-Code
Define security and access control declaratively using `protect`, `verify`, `restrict`.

### 2. Persistent State
Smart contracts with `persistent storage` that maintains state across executions.

### 3. Type Safety
Strong typing with entities, contracts, and type inference.

### 4. Reactive Programming
`watch` keyword for automatic reactions to state changes.

### 5. Dependency Injection
Built-in DI system for testable, modular code.

### 6. VM-Accelerated Execution
Hybrid interpreter/compiler with bytecode VM for performance.

### 7. Blockchain Native
Built-in support for transactions, events, and smart contracts.
